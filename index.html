개인정보처리방침

오밥완은 메뉴 룰렛 제공에 필요한 최소한의 정보만 브라우저에서 처리합니다.

1. 직접 수집하는 정보

오밥완은 회원가입이나 자체 서버를 운영하지 않으며, 입력한 제목과 메뉴는 URL 및 현재 브라우저에서만 처리됩니다.

2. 외부 서비스

광고와 제휴 기능을 위해 카카오 애드핏, 쿠팡 파트너스, 텐핑을 사용합니다. 각 사업자는 광고 제공·성과 측정을 위해 쿠키, 접속 기록, 기기 정보를 처리할 수 있습니다.

3. 공유 기능

공유 버튼은 브라우저의 Web Share 또는 클립보드 기능을 사용하며, 사용자가 실행하기 전에는 외부로 전송하지 않습니다.
